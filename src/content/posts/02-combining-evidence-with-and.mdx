In the [previous post](#/p/sequential-decision-making) we turned a single question
into a **bitmask**: one integer recording which secrets it keeps. That was setup.
The point of a representation is what it lets you *compute*, and here the payoff is
immediate — combining evidence becomes one machine instruction.

## A belief is a mask

> Placeholder — define the running belief as a set / mask.

At any moment the agent's belief is just the set of secrets still consistent with
everything it has heard — and a set over a fixed roster of secrets is a bitmask.
We'll start from a non-trivial belief (say, *warm-blooded animals*) rather than
the whole roster, so the very first combination has something to do.

## Combining evidence is bitwise AND

> Placeholder — derive AND as the intersection / meet.

A new answer is itself a mask: the secrets consistent with that answer. To fold it
into the belief, keep only the secrets consistent with **both** — the
**intersection** of the two sets. On bitmasks, intersection *is* the bitwise
**AND**: a secret survives only where it has a `1` in every mask.

## Two things to notice

> Placeholder — monotonicity and the no-resurrection property.

- **It only ever shrinks.** AND can clear a bit but never set one. Evidence
  eliminates; it never resurrects a secret you've already ruled out. The survivor
  count is monotonically non-increasing.
- **It can bottom out.** Fold in contradictory answers and the belief reaches the
  empty mask — zero survivors. That's the representation telling you your evidence
  is inconsistent, which is a feature, not a crash.

This little AND is the **meet** of a semilattice — the algebraic structure that
makes "combine what I know" well-defined.

## What's next

By now the picture is simple: your belief is a bitmask, each answer is another mask, and combining them is just AND. Ask more questions and the set of live possibilities can only shrink — never grow, never reappear.

But there's a loose end worth pulling. We applied those questions in some particular order. If you'd asked them the other way around — mammal before flight instead of after — would you have landed on a different answer?

If the order can change where you end up, then this isn't really a *belief*; it's bookkeeping that happens to depend on how you got there. If it can't, the mask is something stronger. Next post: which one it is — and why the answer is the property the whole construction rests on.
